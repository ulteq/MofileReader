10 Remove leading directory components from a path, like [tar(1)](http://linuxcommand.org/man_pages/tar1.html)'s `--strip-components` option
11 
12 ```javascript
13 var stripDirs = require('strip-dirs');
14 
15 stripDirs('foo/bar/baz', 1); //=> 'bar/baz'
16 stripDirs('foo/bar/baz', 2); //=> 'baz'
17 stripDirs('foo/bar/baz', 999); //=> 'baz'
18 ```

32 ```
33 
34 ### stripDirs(*path*, *count* [, *option*])
35 
36 *path*: `String` (A relative path)
37 *count*: `Number` (0, 1, 2, ...)
38 *option*: `Object`
39 Return: `String`
40 
41 It removes directory components from the beginning of the *path* by *count*.
42 
43 ```javascript
44 var stripDirs = require('strip-dirs');
45 
46 stripDirs('foo/bar', 1); //=> 'bar'
47 stripDirs('foo/bar/baz', 2); //=> 'bar'
48 stripDirs('foo/././/bar/./', 1); //=> 'bar'
49 stripDirs('foo/bar', 0); //=> 'foo/bar'
50 
51 stripDirs('/foo/bar', 1) // throw an error because the path is an absolute path
52 ```
53 
54 If you want to remove all directory components certainly, use [`path.basename`](http://nodejs.org/api/path.html#path_path_basename_p_ext) instead of this module.
55 
56 #### option.narrow
57 
58 Type: `Boolean`
59 Default: `false`
60 
61 By default, it keeps the last path component when path components are fewer than the *count*.
62 
63 If this option is enabled, it throws an error in such case.
64 
65 ```javascript
66 stripDirs('foo/bar/baz', 9999); //=> 'baz'
67 
68 stripDirs('foo/bar/baz', 9999, {narrow: true}); // throws an error
69 ```

71 ## CLI

79 ### Usage
80 
81 ```sh
82 strip-dirs <string> --count(or -c) <number> [--narrow(or -n)]
83 ```
84 
85 Or, use with pipe(`|`):
86 
87 ```sh
88 echo <string> | strip-dirs --count(or -c) <number> [--narrow(or -n)]
89 ```
90 
91 ### Flags
92 
93 ```sh
94 --count, -c: Number of directories to strip from the path
95 --narrow, -n: Disallow surplus count of directory level
