Return true if a file path is absolute.

Based on the isAbsolute utility method in express.

Install with npm

npm i is-absolute --save

Usage

var isAbsolute = require('is-absolute');
console.log(isAbsolute('a/b/c.js'));
//=> 'false';

Running tests

Install dev dependencies.

npm i -d && npm test
