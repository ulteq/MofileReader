3 unbzip2-stream
4 ===
5 streaming bzip2 decompressor in pure JS for Node and browserify.
6 
7 Buffers
8 ---
9 When browserified, the stream emits instances of [feross/buffer](https://github.com/feross/buffer) instead of raw Uint8Arrays to have a consistant API across browsers and Node.
10 
11 Usage
12 ---
13 ``` js
14 var bz2 = require('unbzip2-stream');
15 var fs = require('fs');
16 
17 // decompress test.bz2 and output the result
18 fs.createReadStream('./test.bz2').pipe(bz2()).pipe(process.stdout);
19 ```
20 
21 Also see [test/browser/download.js](https://github.com/regular/unbzip2-stream/blob/master/test/browser/download.js) for an example of decompressing a file while downloading.
22 
23 Or, using a <script> tag
24 ---
25 
26 ```
27 <script src="https://npm-cdn.info/unbzip2-stream/dist/unbzip2-stream.min.js"></script>

30  // now pipe stuff through it (see above)
